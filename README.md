# Compilation

Just run the build.sh file from the repository root

# HOW TO MAKE GAMES

Have the binary in your desired game folder and run
```bash
./polar init (MAIN SCRIPT NAME).lua
```
It will create a data folder with (MAIN SCRIPT NAME).lua inside, and start the Polar Interactive Shell System (PISS).
PISS contains everything you need to create a game.
PISS has commands like `edit`
Insert the rest of the PISS explaination here

# HOW TO RUN GAMES
If the polar binary is not in the games root folder, add it.
then run
```bash
./polar run data/main.lua
```

Replace main.lua with whatever lua file is the main lua file.
